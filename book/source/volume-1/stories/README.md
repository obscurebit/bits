# Book-Only Story Edits

Place book-edition story overrides in this directory using the same filename as the web story, for example:

`2026-06-02-the-substitutes-compact.md`

When an override exists, the book builder uses its title and body for the manuscript/PDF while preserving the original web post metadata, generation reference, slug, and QR target from `docs/bits/posts`.

This lets the public site remain the daily generated archive while the book can receive editorial revisions.
