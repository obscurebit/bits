## 3. NYC's First Subway, A One Block Pneumatic Tunnel Below Broadway - Untapped New York

In 1870, inventor Alfred Ely Beach developed the first NYC subway line running on pneumatics which ran for a block underneath Broadway in lower Manhattan.
